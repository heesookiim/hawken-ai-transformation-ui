Available API Endpoints

Generate Analysis

POST /api/generate
Content-Type: application/json

{
  "companyName": "Company Name",
  "companyUrl": "https://company-website.com"
}

Generate Company-Specific Content

POST /api/analysis/:companyId/generate
Content-Type: application/json

{
  "prompt": "Generate a detailed implementation plan for..."
}

API Usage Example

fetch('https://hawken-ai-transformation-27d8ee0ab1a5.herokuapp.com/api/generate', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    companyName: "Airbnb",
    companyUrl: "https://www.airbnb.com"
  })
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));

For more information, contact the administrator or refer to the API documentation.
